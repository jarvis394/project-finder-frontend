### Как запустить проект

1. Склонируйте репозиторий
    ```bash
    git clone https://gitlab.com/project-finder-team/project-finder-frontend.git
    ```
2. Установите зависимости
    ```bash
    npm install  # or npm i
    ```
3. Запустите проект
    ```bash
    npm run dev
    ```
